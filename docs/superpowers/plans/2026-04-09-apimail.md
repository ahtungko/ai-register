**Architecture:** Introduce a dedicated `ApiMailProvider` class under `util/providers`, register it in the mail factory, and extend config normalization to support config values plus environment overrides. Keep behavior aligned with existing provider return shapes so `openai` and `grok` flows work unchanged.

**Tech Stack:** Python, requests, unittest, existing mail provider abstraction
